# QuickOpen

### 简介
使用 uTools 工具快速打开历史项目, 支持 JetBrains、vscode, 后续会添加 Xcode的支持

该项目仅支持 macOS，暂不考虑支持Linux、window系统，如果你有需要可以 fork 修改，或者自主开发

项目地址：[https://github.com/JoysKang/QuickOpen](https://github.com/JoysKang/QuickOpen)
